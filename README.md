## Website Pages

| Page | Description |
|------|--------------|
| **Home (index.html)** | Main landing page showcasing the AutoDeals brand and featured cars. |
| **All New Cars** | Displays the latest car models available in the market. |
| **Used Cars** | Lists affordable pre-owned vehicles with key details. |
| **Car Comparison** | Compare specs, prices, and features between different car models. |
| **Car Finance** | Explains financing options and calculators for easy car purchase. |
| **Car Import** | Provides information about importing vehicles from international markets. |
| **Car Insurance** | Shows available insurance plans and their coverage. |
| **Car Tracker** | Vehicle tracking and monitoring service details. |
